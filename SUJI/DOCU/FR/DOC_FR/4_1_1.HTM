4.1.1 Double redraw de certains desktopsSuJi
SuJi demande à l'AV-Server, qui offre ces options, quelle fonte doit être utilisée dans la fenêtre du fichier. Beaucoup d'AV-Server informent tous les programmes, si ils ont changé les réglages dans ce cas.

Quand ce n'est pas le cas, alors SuJi demande par précaution à chaque redraw des fenêtres, quelle est la fonte actuelle. Ainsi SuJi est toujours avec la fonte actuelle. Sans doute juste après le redraw, de telle sorte que, quand la fonte a changé, le redraw doit se répéter.

Cela ne peut être changé que dans l'AV-Server. Alors si nécessaire demandez à l'auteur (ou l'AV-Server) de le faire. (Jinnee le supporte depuis la version 1.02)
