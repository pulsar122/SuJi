2.2 Déclarer SuJi seulement pour JinneeSuJi
Copiez ensembles SuJi.gtp et SuJi.rsc dans un quelconque dossier.

Cliquez dans Jinnee sur la ligne "Applications ..." dans le menu "Divers".

Choisissez dans la boîte liée le 4. Inscription en haut (généralement: "MGSEARCH.APP")

A moitié hauteur de la boîte apparaît dans la boîte de sélection "(Finder)"

Cliquez sur "Autre"

Choisissez "SuJi.gtp"

Activez la boîte "AV: Kbshift" (en haut à droite)

Placez la ligne de commande sur "%sw"

Placez le choix sur "Start"

Cliquez sur "OK"

Puis "Divers: sauver bureau"
