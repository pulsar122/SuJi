Programmer
----------

You need:

- PureC

- Pure C Startup Code from Behne und Behne Software

- PCGEM Library from Martin Els„sser ( >  Version 0.9.2 )

  http://acspro.atari.org/

  If you can't find a version higher than 0.9.2 ask me.
  gerhard_stoll@gmx.de

- PCTOS Library form Martin Els„sser ( >= Version 0.6.0 )

  http://acspro.atari.org/

  If you use the 0.6.0. There is a bug in acstos.h. Change the following
  line:
    Old:
    #define FA_ATTRIB   (FA_SUMDIR|FA_READONLY|FA_HIDDEN|FA_SYSTEM)
    New:
    #define FA_ATTRIB   (FA_SUBDIR|FA_READONLY|FA_HIDDEN|FA_SYSTEM)

- ResourceMaster version 3.5 or higher
